#grepg
>The ruby client for greppage

`grepg` is a ruby client for greppage. It is packaged as a gem.

###Install
To install `grepg`
```gem install grepg`

###Examples
Get all items/microdoc for a topic on evidanary's collection.
```grepg evidanary unix```

Search across items
```grepg evidanary unix -g tcpdump```
